# Golang表示各种进制的数值

整型可以表示为十进制(decimal)、八进制(octal)、十六进制(hex)和二进制(binary)

```go
0xF // 十六进制表示，使用0x或者0X开头
0XF // 每一个字符可以表示4位

017  // 八进制表示，使用0，0o，0O开头
0o17 // 每一个字符可以表示3位
0O17

0b1111 // 二进制表示，必须0b,0B开头
0B1111 // 每一个字符表示1位

15 // 十进制表示
```
