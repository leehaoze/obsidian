# In-Context Learning

LLM之前的各类模型，其能力都是受限于其数据集，只能解决数据集中见过的问题。当一个模型训练好以后，模型内的参数都已经固定下来，其能够解决的问题类型也就固定下来了。

比如一个识别汽车品牌的模型，可以通过图片识别出图中的汽车品牌，某一天一个新的品牌出现了，那这个模型就无法识别出新出现的品牌。

而LLM可以在不更新模型参数的情况下，通过输入中提供的示例来学习新的技能或解决新任务的能力，这种能力被称为上下文学习。
